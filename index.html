Connect Airtable

Henry Watch Tracker reads directly from your Airtable base. Nothing is stored anywhere except this browser.

Need a token? In Airtable: Account → Developer Hub → Personal access tokens. Scopes: data.records:read (add data.records:write only if you want write-back). Access: this base only.

TMDb key is free at themoviedb.org → Settings → API. Used only to fetch a poster image when a title has none in Airtable — nothing is written back to TMDb.

Install App

Add Henry Watch Tracker to your home screen so it opens full-screen, like a native app — no browser bar, no address bar.

1
Open this site in Chrome

Make sure you're using Chrome on Android, not another browser.

2
Tap the ⋮ menu

Top-right corner of Chrome.

3
Tap "Add to Home screen" or "Install app"

Chrome may show either label depending on your version.

4
Confirm

An app icon appears on your home screen. Tap it to launch full-screen.

If you see an automatic "Install" banner at the top of Home, that does the same thing in one tap.
